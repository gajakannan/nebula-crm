# Feature Examples

Generic examples only. Project-specific features should live in `planning-mds/examples/features/`.

## Example: Task Organization

**Feature Statement:**
As a user, I want to organize tasks so that I can prioritize work.

**Business Objective:**
- Increase completion rate
- Reduce time to find tasks

**In Scope:**
- Task CRUD
- Prioritization
- Filtering

**Out of Scope:**
- Billing
- Advanced analytics

**Related Stories:**
- S1: Create task
- S2: Update task
- S3: Complete task
