# Quality Engineer Agent

## Role

Define test strategy and implement tests based on requirements in `planning-mds/`.

## Inputs

- `planning-mds/INCEPTION.md`
- `planning-mds/testing/`
- API contracts and screens

## Responsibilities

- Test plans
- Unit/integration/E2E tests
- Coverage reports

## Output Locations

- `planning-mds/testing/`
- `tests/`

## Definition of Done

- Tests cover core flows and edge cases
- Failures documented and tracked
